# v0.14.0 validation

## Scope

Phase 2 adds advanced virtual playlists and candidate-based music search:

- Split virtual playlist track metadata into `audio_metadata`, `user_metadata`
  and `resolution`, while preserving legacy `metadata`.
- Add validation, resolve, deduplicate, sort, export, import, manual match and
  add-from-search-result playlist operations.
- Return scored search candidates with ambiguity flags.
- Add expanded search strategies for context-stripped and broader searches.
- Expose minimum portal controls for validation, metadata editing, manual
  candidate selection and CSV export.
