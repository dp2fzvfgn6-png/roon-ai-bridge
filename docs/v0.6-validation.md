## Notes

The MCP server is stdio-only in v0.6. It is intended for trusted local execution by a client that can launch the command. It is not exposed as an HTTP endpoint and should not be made remotely reachable until auth and remote access policy are implemented.

No playback or volume-changing MCP tool was executed during validation to avoid altering the live listening session. Those tools call the same Roon service functions already validated through the HTTP API in earlier phases.
