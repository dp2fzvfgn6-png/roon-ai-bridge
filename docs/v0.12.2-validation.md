## Temporary Playlist Validation

A temporary playlist with prefix `roonia_test_` was created and deleted during
validation. No playback was started.

Validated behavior:

- `include_tracks=false` returned playlist metadata without a `tracks` array.
- Detail read returned `track_count=3`.
- `include_tracks=true&limit=2&offset=0` returned `returned_count=2` and
  `has_more=true`.
- `offset=99` returned `returned_count=0` and `has_more=false`.
- Updating the last track with `position=1` moved it to the first position.
- Cleanup deleted the temporary playlist successfully.
