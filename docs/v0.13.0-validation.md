# Roon AI Bridge v0.13.0 Validation

Date: 2026-07-09

## Scope

v0.13.0 adds RoonIA zone presets, portal-only virtual zones and configurable
safe volume limits.

- Zone presets persist name, description, enabled state, grouping, target
  volumes, playback action and queue action.
- Presets can be dry-run or applied without starting music or replacing queues.
- Volume limits can target output id, zone id, output name, zone name or global.
- Scheduled limits support day lists and windows that cross midnight.
- Overlapping schedules for the same target are rejected.
- `roon_change_volume` and preset application evaluate the active safe limit.
- The portal can list/create/edit/delete/apply presets and manage/evaluate
  volume limits.

## Roon Virtual Zone Decision

RoonIA does not create real Roon zones for presets. The Roon SDK exposes
Transport operations for existing zones/outputs and extension-provided
status/settings/device-control services, but no supported way for an extension
to create an additional visible playback zone without being a real output/device
integration. Presets are therefore RoonIA entities shown as virtual zones only
inside RoonIA/portal.
