## Notes

Use a long random token before exposing the API through Nginx Proxy Manager:

```bash
openssl rand -hex 32
```

Then set:

```env
ENABLE_AUTH=true
API_TOKEN=<generated-token>
```

Clients must send:

```http
Authorization: Bearer <generated-token>
```

MCP remains stdio-only and local/trusted in v0.7. This HTTP API token does not turn MCP into a remote service.
