# v0.10 Validation

v0.10 adds SQLite-backed virtual playlists, richer library items and native
Roon zone grouping.

## Automated Tests

```bash
pnpm run test
pnpm run build
```

Validated locally:

- Legacy JSON playlists migrate to SQLite.
- Docker and package engines use Node.js 24 for native `node:sqlite`.
- Playlist and track CRUD, replacement and reordering work transactionally.
- Library items include normalized song metadata and cover references.
- Grouping sends the primary output first so its queue is preserved.
- Grouping is rejected when outputs are not mutually compatible.
- Grouping only succeeds after all selected outputs appear in one zone.
- Ungrouping only succeeds after every output becomes independent.
- The complete suite passes.
