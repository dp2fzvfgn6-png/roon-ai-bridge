## MCP Contract Validation

`tools/list` exposed the expected updated schemas:

- `roon_list_zones` includes `include_image_data`.
- `roon_list_virtual_playlists` includes `include_tracks`, `limit`, `offset`,
  `track_limit`, and `track_offset`.
- `roon_search_media` includes `include_images`, typed `types`, `count`, and
  `source_preference`.
- `roon_resolve_virtual_playlist` is registered.

Live read-only MCP calls:

- `roon_list_zones` with default arguments did not return `image_data_url`.
- `roon_search_media` for `Radiohead Everything In Its Right Place` returned a
  `media_` result ID, did not return `core_connected`, and did not return
  `image_data_url`.
- `roon_get_media_details` with the returned `result_id` returned matching
  media details and did not return `core_connected`.
- `roon_list_virtual_playlists` with `include_tracks=false`, `limit=2`, and
  `offset=0` did not return track arrays, did return `track_count`, and
  respected `limit=2`.
