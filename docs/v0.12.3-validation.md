# Roon AI Bridge v0.12.3 Validation

Date: 2026-07-09

## Scope

v0.12.3 is a narrow stabilization release:

- Makes `roon_update_virtual_playlist_track` atomic when validation fails.
- Ensures invalid `position` values are rejected before track fields are written.
- Publishes the ChatGPT App widget as `ui://roon-ai-bridge/control-v5.html`
  to avoid stale widget metadata.
- Adds an HTTP MCP manifest regression test for final tool schemas,
  descriptions and widget URI.

No playback, queue mutation or volume behavior was changed.

## MCP Manifest Validation

`tools/list` through the deployed MCP endpoint exposed:

- `roon_get_virtual_playlist` schema includes `playlist_id`,
  `include_tracks`, `limit` and `offset`.
- `roon_get_virtual_playlist` uses widget URI
  `ui://roon-ai-bridge/control-v5.html`.
- No tool reused the `roon_status` description.

This explains the stale ChatGPT behavior seen after v0.12.2: the backend
manifest was fixed, but the ChatGPT App could keep old tool/widget metadata
cached. v0.12.3 bumps the widget URI and documentation now calls out refreshing
the app configuration and starting a new conversation after metadata changes.
