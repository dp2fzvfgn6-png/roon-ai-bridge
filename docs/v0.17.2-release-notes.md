# RoonIA v0.17.2 Release Notes

Date: 2026-07-17
Release channel: stable (`main`)
Source channel: beta

## Summary

v0.17.2 promotes the tested beta line that followed v0.17.1. It strengthens
model-created playlists and Roon catalog navigation, expands the portal's
playback and update experience, and moves embedded widgets to self-contained
v18 artwork payloads.

## User-visible changes

- Model-created playlists use batched preflight resolution, reject ambiguous
  or unintended recordings, deduplicate tracks and report any final shortfall.
- Search and detail views retain exact Roon catalog identity, follow native
  continuations and expose complete streaming discographies and track lists.
- The portal adds recent activity history, progressive concurrent search,
  rotating playlist collages, total playlist duration and clearer action
  notifications.
- System settings add daily update checks, persistent update availability,
  guided beta-exit choices and a persistent diagnostics mode.
- Widget v18 embeds bounded server-loaded artwork privately in component
  metadata, avoiding dependency on public image routing.

## MCP and widget compatibility

The canonical MCP v2 intent boundary remains in place. Search, playlist and
widget results now include the additional verified state needed by the new
portal and component behavior. The current resources use the cache-busted
`ui://roon-ai-bridge/v18/` namespace.

After deployment, refresh the ChatGPT app and start a new conversation so it
reloads tool metadata and widget resources.

## Upgrade and rollback

Stable installations follow `main`; installations with beta updates enabled
follow `beta`. The LXC updater preserves runtime data and configuration while
rebuilding from the selected Git ref. Existing databases are migrated in place
for activity history and the latest playlist metadata.

Before rollout, back up `/opt/roon-ai-bridge/data` and the LXC `.env`. Rollback
should select the previous validated tag and use the documented LXC update
procedure.

See [Update Existing LXC](update-lxc.md),
[v0.17.2 Validation](v0.17.2-validation.md),
[Deterministic Search Validation](deterministic-search-validation.md) and
[Widget v18 Embedded Artwork Validation](widget-v18-validation.md).
